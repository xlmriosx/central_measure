# Measures central tendence

### median = $$ \frac{1}{N} \sum_{i=1}^N x_i  $$ 

### mediana(unpar) = $$ x_{(n+1)/2}^{\text{ordered}} $$

### mediana(par) = $$ \frac{x_{n/2}^{\text{ordered}} + x_{n/2+1}^{\text{ordered}}}{2} $$

### moda = $$ x_k $$ where $$ \text{Freq}(x_k) = \max{(\text{Freq}(x_i))} $$
